## 🖥️ Run locally (PICO-8)

1. Download `astro.p8.png` (or `astro.p8`)
2. Open PICO-8
3. `LOAD astro.p8.png`
4. `RUN`

---

## 🎮 Controls

- Arrow Keys → Move  
- Z → Shoot  

---

## 📜 Rules

- Destroy incoming enemies and survive waves as long as possible
- Steal their energy to become more powerfull, but take one hit and loose it all
- Avoid enemy ships and projectiles  
- The game ends when you lose all lives or complete all waves

---

## 🧠 About the Project

Astro: Demake is based on a shoot’em up project originally developed in Godot, which already included most of the core mechanics and systems.

This version is a reinterpretation built in PICO-8 as part of a learning process, focusing on adapting those mechanics within a more constrained environment.

The goal is to explore core gameplay systems like movement, shooting, and enemy behavior while working within the limitations of PICO-8.

Expect ongoing changes and experimentation as development continues.
